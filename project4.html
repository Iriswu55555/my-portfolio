Personalized Mandarin Learning

1:1 Tutoring as Learning Experience Design

Overview

Designed and delivered personalized Mandarin learning experiences for adult English speaker. I applied instructional design principles to create custom curriculum tailored to learner’s goals, preferences, and pace.

Tools

Gooogle classroom , Zoom, Google doc

Learning Objectives

  • Build confidence in speaking and listening
  • Develop the ability to have a daily conversation using target structures
  • Improve tone and pronunciations

ADDIE model

A
Analysis
  • Conducted interviews to assess learner’s goals and prior knowledge.
    Beginners: understand daily conversation, but often struggle with tones and pronunciations.
  • Identified learner-specific needs: visual memory support.
    To learn daily conversation; no test nor time constraints.

D
Design
  • Designed a lesson structure using visual aids and conversational practice.
    Included culturally relevant dialogues to maintain learner interest. ex.food, music
  • Selected multimedia materials tailored for visual learners.
    Flashcards, videos, songs, and daily-use scenario animations.

D
Development
  • Created weekly learning kits:Interactive slide decks with native audio.
  • Designed take home assignment

I
Implement
  • Facilitated live 1-on-1 sessions with visuals, digital whiteboards (Jamboard), and role-play.
